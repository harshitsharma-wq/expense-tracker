# 💰 Smart Expense Tracker

## 📌 Project Overview

Smart Expense Tracker is a web-based application that helps users manage their daily income and expenses efficiently. Users can add income, record expenses, categorize spending, view expense summaries, check remaining balance, and analyze expenses through graphical visualization.

The project is built using HTML, CSS, and JavaScript for the frontend, while MongoDB and Node.js/Express are used for backend data storage and management.

---

## 🚀 Features

* Add and manage income
* Add daily expenses with:

  * Title
  * Amount
  * Category
  * Date
* Automatic expense calculation
* Remaining balance calculation
* Displays **"Insufficient Balance"** when expenses exceed income
* View all expense records
* Expense visualization using Pie Chart
* Reset all expense records
* MongoDB database integration for data storage

## 📊 Working

1. User enters income.
2. User adds expenses with category and date.
3. Expense data is stored in MongoDB.
4. Total expenses are calculated automatically.
5. Remaining balance is displayed.
6. Users can view all expenses and expense charts.
7. Users can reset all stored expense data.
